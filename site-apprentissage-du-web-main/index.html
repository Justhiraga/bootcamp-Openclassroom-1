Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le World Wide Web (WWW) ou plus communément appeler Web peut être vu comme une toile : chaque page est un nœud de la toile et le fil entre deux nœuds représente le lien hypertexte entre deux pages

Internet

Internet est le résau qui permet de communiquer entre ordinateur, sur ce réseau ont y trouve plusieurs services : le web ; les E-mails ; les newsgroups ; le FTP ; ETC...

Cloud

Le cloud est un service permettant de stocker des données sur des serveurs via internet, ex: photos, vidéos, musiques, fichiers, etc...

W3C

organisme fondé par Tim Berners-Lee chargé de promouvoir la compatibilité des technologies du WWW telles que: HTML5, HTML, XHTML, XML, RDF, SPARQL, CSS, XSL, PNG, SVG, MathML et SOAP.

Partie 2 - Distinguez les langages du Web

Langages client

HTML, CSS et JavaScript

Navigateurs

Chrome, mozilla, Edge

Langages serveur

Python , Ruby, C#, C++ et PHP

Frameworks

il s’agit de bibliothèques élaborées qui permettent de structurer l’entièreté de son code Ils sont utilisés notamment pour des sites web plus importants, qui nécessitent beaucoup de code

Bases de données

Les bases de données servent a enregistrer des information sur le stie ex: combiens de visiteur sur le site , combiens de comptes crées, nombres de produits afficher, etc...

Site responsive

consiste a rendre un site adaptable a tous les appareils: tablettes, smartphones, etc...

Application mobile

pour les application native ont utilise des langages mobiles spécifiques à Android et IOS (par exemple : Kotlin, Java ou Swift.)

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Les serveurs distribuent les sites web depuis les datacenters

Adresse IP

L’adresse IP permet de retrouver un ordinateur sur la planète

Nom d’hôte et DNS

Le nom d’hôte (ex : google.com) est plus facile à retenir que l’IP, C’est pour cela que l’on fait une traduction via un annuaire, appelé le DNS.

Protocoles

Les protocoles indiquent comment les machines doivent communiquer entre elles. Il existe des protocoles de bas niveau (TCP, UDP) et des protocoles de haut niveau (HTTP, HTTPS, FTP, SMTP)

Partie 4 - Identifiez les métiers du web

Développeurs

Ceux qui développent les sites web Ils utilisent les langages de programmation pour coder le site ou l’application

Product Managers et chefs de projets

Ceux qui coordonnent la production ou l’évolution d’un produit web Ils ne codent pas directement, mais sont en lien direct avec les équipes techniques Ils ont une vision globale du produit et des fonctionnalités à développer, et assurent la bonne gestion du projet.

Designers

Ceux qui font en sorte que le site soit le plus simple d’utilisation possible, les professionnels du design et de l’ergonomie des pages web ou des applications.

Testeurs

un professionnel spécialisé en informatique chargé de réaliser des tests et des vérifications sur différents logiciels afin de garantir leur bon fonctionnement

Techniciens informatique

Ceux qui gèrent l’infrastructure du web, ce sont les professionnels des SI (systèmes d’information). Ils ont la responsabilité des serveurs, de leur configuration, sans eux les développeurs et chefs de projets ne pourraient pas travailler, et le site web ne répondrait tout simplement pas.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

langage informatique permettant de mettre en forme des pages web (HTML ou XML)

JavaScript

Front-End

langage de programmation qui permet de créer du contenu mis à jour de façon dynamique, de contrôler le contenu multimédia, d'animer des images, et tout ce à quoi on peut penser.

JavaScript

Back-End

JavaScript back-end sert a écrire du code côté serveur, ce code est en charge de la logique côté serveur d'une application, y compris les interactions avec la base de données.

PHP

Back-End

langage informatique open source permettant de créer des pages web dynamiques du côté serveur.

Java

Back-End

Java est un langage de programmation largement utilisé pour coder des applications web, Java est un langage multiplateforme, orienté objet et centré sur le réseau, qui peut être utilisé comme une plateforme à part entière.

Python

Back-End

Python est un langage de programmation interprété, multiparadigme et multiplateformes. Il favorise la programmation impérative structurée, fonctionnelle et orientée objet.

C#

Back-End

langage de programmation orientée objet, commercialisé par Microsoft depuis 2002 et destiné à développer sur la plateforme Microsoft .NET

SQL

Bases de données

Le langage de requête structurée (SQL) est un langage standard pour la création et la manipulation de bases de données. MySQL est un programme de base de données relationnelle qui utilise des requêtes SQL.

Boîte à outils du développeur

  • IDE comme VS Code : IDE signifie « environnement de développement intégré », c'est-à-dire que tous les éléments nécessaires à l'écriture, à la compilation et au débogage sont intégrés. VS Code peut avoir les mêmes fonctionnalités via des extensions, mais sémantiquement, ce n'est pas un IDE.
  • Github : GitHub est une plateforme basée sur le cloud où vous pouvez stocker, partager et travailler avec d'autres pour écrire du code.
  • Librairies / bibliothèques : Une librairie est un ensemble de fonctions et de classes déjà codées dans un langage spécifique. Ainsi, le développeur (ou programmeur) peut aller y piocher en fonction de ses besoins. Inutile de coder toutes les fonctions.
  • Frameworks : il s’agit de bibliothèques beaucoup plus élaborées, qui permettent de structurer l’entièreté de son code, comme un kit de construction ! Ils sont utilisés notamment pour des sites web plus importants, qui nécessitent beaucoup de code. ex: Angular → JavaScript / Django → Python / Zend → PHP
